Project Description

We created a Frogger program that has a player who is given five lives. Whenever they hit an enemy, they lose a life and are sent back to their starting point. There are three enemies who are the same size, but move at different speeds and directions. The player has to get to the other side to win. When the player has lost all of their lives, the enemies stop moving and the game is over. We used Canopy and GitHub to create this game.


Project Reflection

The most challenging part of this project was trying to get the player to return to its starting point when it hit an enemy. What I did to solve this problem was put a black square directly under the player at its starting point. You cannot see the square but this made it much easier for the player to return to an exact position. I got this idea from Lab 16 when the rocket had to return to the player. Things I would do differently for the project include making enemies that moved up and down and making text appear that said when the game was over. I would still make the same project because I am pleased with the final game. I did the project in the to-do list order and I used example code from other projects to complete the program.
