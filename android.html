Project Description

Nicki Hilker and I made a movie trivia game where during the first round, the player sees a picture of an actor and chooses the actor’s name from four multiple choice answers. The player learns if they are correct or incorrect and then they go to the next question. There is a second round where people can match actors to their movies. It is in the same format as Part 1. We used MIT App Inventor and iPads to do this project.


Kendall's Project Reflection

The hardest part of this project for me was figuring out how to add more screens and making a button that would go from one screen to the next. It was also hard to remember which answer it was when I was doing the coding for that particular part. If we’d had more time, we could have made some sort of scoring system that would count your correct answers for you. If I could have done something different, I would have added more trivia questions or put in little fun facts about the movies and actors. The easiest part of the project was deciding what to do. Most people know who the actors are and what movies they were in, but they can still have fun playing the game.
